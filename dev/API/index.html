Types Used to Define a Model

Hephaestus.Model — Type
struct Model

A type representing the finite element model of a structure of interest.

  • name: Name of the model

  • nodes: Nodes of the model

  • sections: Sections of the model

  • materials: Materials of the model

  • elements: Elements of the model

  • concloads: Concentrated loads of the model

  • distloads: Distribution loads of the model

source
Hephaestus.Node — Type
struct Node

A type representing a node in a finite element model.

  • ID: Identification tag

  • x: $x$-coordinate

  • y: $y$-coordinate

  • z: $z$-coordinate

  • u_x: Is the node restrained agaist translation in the global $x$-direction?

  • u_y: Is the node restrained agaist translation in the global $y$-direction?

  • u_z: Is the node restrained agaist translation in the global $z$-direction?

  • θ_x: Is the node restrained agaist rotation about the global $x$-axis?

  • θ_y: Is the node restrained agaist rotation about the global $y$-axis?

  • θ_z: Is the node restrained agaist rotation about the global $z$-axis?

source
Hephaestus.Material — Type
struct Material

A type representing a material in a finite element model.

  • ID: Identification tag

  • E: Young's modulus, $E$

  • ν: Poisson's ratio, $\nu$

  • ρ: Density, $ρ$

source
Hephaestus.Section — Type
struct Section

A type representing a section in a finite element model.

  • ID: Identification tag

  • A: Cross-sectional area, $A$

  • I_zz: Moment of inertia about the local $z$-axis, $I_{zz}$

  • I_yy: Moment of inertia about the local $y$-axis, $I_{yy}$

  • J: Torsional constant, $J$

source
Hephaestus.Element — Type
struct Element

A type representing an element in the finite element model.

  • ID: Identification tag

  • node_i: Node ($i$)

  • node_j: Node ($j$)

  • section: Section

  • material: Material

  • ω: Orientation angle, $\omega$

  • releases_i: End releases at node ($i$)

  • releases_j: End releases at node ($j$)

  • L: Element length, $L$

  • γ: Subtransformation matrix, $\gamma$

  • Γ: Transformation matrix, $\Gamma$

  • k_e_l: Elastic stiffness matrix in the local coordinate system, $k_{el}$

  • k_e_g: Elastic stiffness matrix in the global coordinate system, $k_{eg}$

  • k_g_l: Geometric stiffness matrix in the local coordinate system, $k_{gl}$

  • k_g_g: Geometric stiffness matrix in the global coordinate system, $k_{gg}$

  • m_l: Mass matrix in the local coordinate system, $m_{l}$

  • m_g: Mass matrix in the global coordinate system, $m_{g}$

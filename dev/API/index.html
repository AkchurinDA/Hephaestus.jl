Types Used to Define a Model

Hephaestus.Model — Type
struct Model

A type representing the finite element model of a structure of interest.

  • nodes: Ordered dictionary that stores the nodes of the model

  • materials: Ordered dictionary that stores the materials of the model

  • sections: Ordered dictionary that stores the materials of the model

  • elements: Ordered dictionary that stores the elements of the model

  • supports: Ordered dictionary that stores the supports of the model

  • conc_loads: Ordered dictionary that stores the concentrated loads of the model

  • dist_loads: Ordered dictionary that stores the distributed loads of the model

  • p_l: Ordered dictionary that stores the fixed-end forces in the local coordinate system

  • p_g: Ordered dictionary that stores the fixed-end forces in the global coordinate system

Hephaestus.Section — Type
struct Section

A type representing a section in a finite element model.

  • A: Cross-sectional area

  • I_zz: Moment of inertia about the local $z$-axis

  • I_yy: Moment of inertia about the local $y$-axis

  • J: Polar moment of inertia

source
Hephaestus.Element — Type
struct Element

A type representing an element in the finite element model.

  • node_i_tag: Unique identifier of the node ($i$)

  • x_i: $x$-coordinate of the node ($i$)

  • y_i: $y$-coordinate of the node ($i$)

  • z_i: $z$-coordinate of the node ($i$)

  • node_j_tag: Unique identifier of the node ($j$)

  • x_j: $x$-coordinate of the node ($j$)

  • y_j: $y$-coordinate of the node ($j$)

  • z_j: $z$-coordinate of the node ($j$)

  • material_tag: Unique identifier of the material

  • E: Young's modulus

  • ν: Poisson's ratio

  • ρ: Density

  • section_tag: Unique identifier of the section

  • A: Cross-sectional area

  • I_zz: Moment of inertia about the local $z$-axis

  • I_yy: Moment of inertia about the local $x$-axis

  • J: Polar moment of inertia

  • ω: Angle that defines the orientation of the local coordinate system of the element

  • releases_i: DOF releases at the node ($i$)

  • releases_j: DOF releases at the node ($j$)

  • L: Length of the element

  • γ: Local-to-global sub-transformation matrix

  • Γ: Local-to-global transformation matrix

  • k_e_l: Element elastic stiffness matrix in the local coordinate system

  • k_e_g: Element elastic stiffness matrix in the global coordinate system

  • k_g_l: Element geometric stiffness matrix in the local coordinate system

  • k_g_g: Element geometric stiffness matrix in the global coordinate system

  • m_l: Element mass matrix in the local coordinate system

  • m_g: Element mass matrix in the global coordinate system

Types Used to Define a Model

Hephaestus.Model — Type
struct Model

A type representing the finite element model of a structure of interest.

  • nodes: Ordered dictionary that stores the nodes of the model

  • materials: Ordered dictionary that stores the materials of the model

  • sections: Ordered dictionary that stores the sections of the model

  • elements: Ordered dictionary that stores the elements of the model

  • springs: Ordered dictionary that stores the springs of the model

  • supports: Ordered dictionary that stores the supports of the model

  • conc_loads: Ordered dictionary that stores the concentrated loads of the model

  • dist_loads: Ordered dictionary that stores the distributed loads of the model

  • p_l: Ordered dictionary that stores the element fixed-end forces in the local coordinate system

  • p_g: Ordered dictionary that stores the element fixed-end forces in the global coordinate system

source
Hephaestus.Node — Type
struct Node

A type representing a node in the model of a structure of interest.

  • ID: Unique identifier

  • x: $x$-coordinate

  • y: $y$-coordinate

  • z: $z$-coordinate

source
Hephaestus.Material — Type
struct Material

A type representing a material in the model of a structure of interest.

  • ID: Unique identifier

  • E: Young's modulus, $E$

  • ν: Poisson's ratio, $u$

  • ρ: Density, $\rho$

source
Hephaestus.Section — Type
struct Section

A type representing a section in the model of a structure of interest.

  • ID: Unique identifier

  • A: Cross-sectional area, $A$

  • I_zz: Moment of inertia about the local $z$-axis, $I_{zz}$

  • I_yy: Moment of inertia about the local $y$-axis, $I_{yy}$

  • J: Polar moment of inertia about the local $x$-axis, $J$

source
Hephaestus.Element — Type
struct Element

A type representing an element in the model of a structure of interest.

  • ID: Unique identifier

  • node_i_ID: Unique identifier of the node $i$ of the element

  • node_j_ID: Unique identifier of the node $j$ of the element

  • material_ID: Unique identifier of the material of the element

  • section_ID: Unique identifier of the section of the element

  • releases_i: DOF releases at the node $i$

  • releases_j: DOF releases at the node $j$

  • ω: Angle that defines the orientation of the element's local coordinate system, $\omega$

  • x_i: $x$-coordinate of the node $i$, $x_{i}$

  • y_i: $y$-coordinate of the node $i$, $y_{i}$

  • z_i: $z$-coordinate of the node $i$, $z_{i}$

  • x_j: $x$-coordinate of the node $j$, $x_{j}$

  • y_j: $y$-coordinate of the node $j$, $y_{j}$

  • z_j: $z$-coordinate of the node $j$, $z_{j}$

  • E: Young's modulus, $E$

  • ν: Poisson's ratio, $\nu$

  • ρ: Density, $\rho$

  • A: Cross-sectional area, $A$

  • I_zz: Moment of inertia about the local $z$-axis, $I_{zz}$

  • I_yy: Moment of inertia about the local $y$-axis, $I_{yy}$

  • J: Polar moment of inertia about the local $x$-axis, $J$

  • L: Length of the element, $L$

  • γ: Local-to-global sub-transformation matrix, $\gamma$

  • Γ: Local-to-global transformation matrix, $\Gamma$

  • k_e_l: Elastic stiffness matrix in the local coordinate system of the element, $k_{e, l}$

  • k_e_g: Elastic stiffness matrix in the global coordinate system, $k_{e, g}$

  • k_g_l: Geometric stiffness matrix in the local coordinate system of the element, $k_{g, l}$

  • k_g_g: Geometric stiffness matrix in the global coordinate system, $k_{g, g}$

  • m_l: Mass matrix in the local coordinate system of the element, $m_{l}$

  • m_g: Mass matrix in the global coordinate system, $m_{g}$
